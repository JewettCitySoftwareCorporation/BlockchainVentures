BPHT5 is a collection of HTML5 Templates to generate Business | Marketing Plans or project documentation. These are set styles and UI actions & animations for each template. There is NO server side or business logic here, just stand alone html which can be deployed or viewed with any standard tools. This collection has been assembeled to be HTML, CSS, JavaScrip nessasary to design and build business plan data to be presentable.

What Is BPHT5:

Business Plans, Project Documentation, Marketing, are all reoccuring efforts and needed in the marketplace. JCSC.biz set out to build a few Business Plans and saw there is sad HTML5 planning templates or such things advailable. Once JCSC.biz had internal templates, it was decided to try and release these efforts as a tool set to help the community.
